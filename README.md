# Getting Started of financial-demo

there are 2 options builder you can choose :
 * gradle
 * maven
 
 to run with gradle :
 gradlew bootRun -P{profiles}
 
 to run with maven :
 mvnw -Dspring.profiles.active={profiles} spring-boot:run
